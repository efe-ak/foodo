Package Overview

This template is built with Bootstrap 3.2.0 and provides multiple UI Kits. The package is structured in two branches:

  • dist - includes the static files (HTML, images, compiled CSS and JavaScript)
  • dev - includes a full development suite based on the latest tools in front-end development like Grunt, node.js templating, LESS, SASS and even a dedicated Vagrant environment so you can start developing in minutes with almost no setup required and zero effort.
